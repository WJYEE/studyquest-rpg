# StudyQuest Overview 🎮

## 1. Project Introduction

StudyQuest는 공부 시간을 캐릭터 성장 경험으로 전환하는 
게이미피케이션 기반 학습 관리 웹 서비스입니다.


## 2. Background

많은 학습자는 장기간 공부 과정에서 즉각적인 보상이 부족하여
동기 유지에 어려움을 겪습니다.

기존 학습 타이머 서비스는 시간 기록 중심으로 동작하며,
학습 과정 자체의 재미와 성장 경험 제공에는 한계가 있습니다.


## 3. Problem Definition

### User Problems

- 공부 시간이 단순 숫자로만 기록됨
- 장기간 학습 동기 유지 어려움
- 자신의 성장 과정을 체감하기 어려움


## 4. Solution

StudyQuest는 다음 방식으로 문제를 해결합니다.

- Stopwatch 기반 학습 기록
- Pixel Character 성장 시스템
- 과목별 Level 시스템
- Reward & Shop 시스템
- 학습 데이터 Dashboard

이를 통해, 학습을 게임처럼 진행하고 기록하며 강한 동기부여를 얻을 수 있습니다.

## 5. Target User

- 대학생
- 자격증 준비생
- 어학 학습자
- 자기계발을 지속하는 사용자


## 6. v1 Goal

LocalStorage 기반 MVP 구현

### Included

- Subject Management
- Study Timer
- Character Level
- Subject Level
- Coin System
- Basic Shop
- Dashboard


### Excluded

- Database
- Mobile App
