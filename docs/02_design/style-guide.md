# StudyQuest Style Guide

## Design Philosophy
- Productivity first
- Cozy pixel RPG
- Warm and calming
- Minimal game elements

## Character
- Cute chibi proportions
- Detailed pixel art
- Friendly expression
- One static base pose
- Layered equipment system

## Color Palette
- Cream background
- Forest green primary
- Warm brown outlines
- Soft pastel accents

## UI
- Rounded cozy panels
- Soft shadows
- Clean typography
- Minimal decorative elements

## Icons
- Pixel-inspired SVG
- Consistent stroke width
- Simple recognizable silhouettes

## Background
- Quiet cream by default
- Diorama only where appropriate
- Never distract from studying

## Animation
- Calm idle motion
- Reward-focused effects
- Respect reduced motion

## Asset Naming
- Use existing TypeScript identifiers
- Match Item.id, HairStyle, RpgScene, etc.
