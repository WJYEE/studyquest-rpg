# Architecture

Structure:

↓

Next.js Application

↓

React State Management

↓

LocalStorage


---


## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS


State:

- Zustand

- Recharts

- LocalStorage

- Understanding frontend development
- Building core user experience
- Fast MVP validation
